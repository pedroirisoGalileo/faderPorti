# FaderPort USB Controller

Este proyecto contiene el código principal para el controlador USB FaderPort. Utiliza bibliotecas para MIDI USB, encoders y debouncing de botones. 

## Requisitos

- Arduino IDE
- Bibliotecas necesarias:
  - USB-MIDI
  - Encoder
  - Bounce2

## Instalación

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `faderPortiUSB.ino` en el Arduino IDE.
3. Asegúrate de que las bibliotecas necesarias estén instaladas.
4. Conecta tu dispositivo Arduino y selecciona el puerto correcto en el IDE.
5. Sube el código al dispositivo.

## Uso

Una vez que el código esté cargado en tu dispositivo, podrás utilizar el controlador FaderPort para enviar mensajes MIDI a tu software de producción musical. Los encoders y botones están configurados para enviar diferentes mensajes MIDI.
